# Shop Process

## Ziel

Der Shop-Prozess dokumentiert, wie Produkte in den Warenkorb, zur Bestellung und zum Checkout gelangen.

## Erste Phase

- Konzept für Produktkatalog und Warenkorb
- Klare Trennung zwischen Endverbraucher und Großhandel
- Keine Checkout-Implementierung im ersten Block

## Spätere Schritte

- Warenkorb-Logik
- Bestellprozess
- Zahlungen
- Versand
- Bestellbestätigungen
