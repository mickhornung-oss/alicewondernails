# Admin Plan

## Ziel

Das System benötigt später einen Adminbereich zur Pflege von Produkten, Bestellungen, Kunden und rechtlichen Inhalten.

## Erste Stufe

- Django Admin als technisches Fundament
- Nutzer- und Rollenverwaltung für Admin-Zugänge
- Basis für spätere eigene Admin-UI

## Nicht im ersten Block

- eigene Admin-Oberfläche
- erweiterte Admin-Dashboards
- Medienmanager mit Upload-Schnittstelle
